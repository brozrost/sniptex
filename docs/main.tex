\section*{Snip\TeX}

Snip\TeX{} je La\TeX{}ový balíček pro vkládání označených úryvků kódu ze zdrojových 
souborů nebo vzdálených repozitářů dostupných přes webové odkazy. 
Extrahuje označené části kódu a sází je jednotným stylem, čímž podporuje reprodukovatelnou a udržitelnou dokumentaci.
